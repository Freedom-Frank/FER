================================================================================
                    Windows 摄像头启动 - 超简单指南
================================================================================

你遇到的问题：在 PowerShell 中 conda 命令不可用

解决方案：使用 Anaconda Prompt（不是 PowerShell）

================================================================================
                              快速步骤
================================================================================

步骤 1：打开 Anaconda Prompt
  按 Win 键 → 搜索 "Anaconda Prompt" → 打开

步骤 2：进入项目目录
  cd /d E:\Users\Meng\Projects\VScodeProjects\FER

步骤 3：激活环境
  conda activate fer

步骤 4：运行摄像头
  python tools\demo_visualization.py --mode webcam --ckpt checkpoints_50epoch\best_model.ckpt

步骤 5：享受实时表情识别！
  - 按 'q' 退出
  - 按 's' 保存截图

================================================================================
                            更简单的方法
================================================================================

方法 1：使用自动脚本（推荐）

  1. 双击: run_webcam_conda.bat
     （会自动查找 conda 并激活环境）

方法 2：在 Anaconda Prompt 中一键运行

  1. 打开 Anaconda Prompt
  2. cd /d E:\Users\Meng\Projects\VScodeProjects\FER
  3. conda activate fer
  4. run_webcam.bat

================================================================================
                              测试命令
================================================================================

在 Anaconda Prompt 中测试：

  # 测试摄像头
  python test_camera.py

  # 测试模型
  python test_model.py

  # 直接运行
  python tools\demo_visualization.py --mode webcam --ckpt checkpoints_50epoch\best_model.ckpt

================================================================================
                            为什么不用 PowerShell？
================================================================================

PowerShell 需要额外配置 conda。使用 Anaconda Prompt 更简单：

  Anaconda Prompt    ✅ conda 自动可用
  PowerShell         ❌ 需要手动配置

================================================================================
                            还是不行？
================================================================================

如果 Anaconda Prompt 也不行，查看详细文档：

  1. WINDOWS_SETUP.md     - Windows 完整配置指南
  2. WEBCAM_GUIDE.md      - 使用指南
  3. QUICK_FIX_WSL_WEBCAM.md - WSL 用户指南

或尝试：
  1. 重新安装 Anaconda: https://www.anaconda.com/download
  2. 使用普通 Python: https://www.python.org/downloads/
